# 快速入门

> 5 分钟从安装到生成第一份 Word 报告

---

## 1. 安装

```bash
pip install -r requirements.txt
```

## 2. 运行

准备好一个模板文件（.docx）和一个数据文件（.xlsx 或 .yaml），执行：

```bash
python -m src --data 数据.xlsx --template 模板.docx --output 输出.docx
```

**简化写法**：

```bash
python -m src -d 数据.xlsx -t 模板.docx -o 输出.docx
```

## 3. 看到结果

命令行输出：

```
读取Excel: 数据.xlsx
上下文统计: 5 个顶级条目
加载模板: 模板.docx
执行渲染...
保存文档: 输出.docx
✓ Render complete: 输出.docx
```

打开 `输出.docx`，所有变量已被数据替换。
