## A. 标签三级层级

docxtpl 中所有标签按前缀分为三个作用域层级：

| 层级 | 前缀 | 能包裹什么 | `{% X %}` 和 `{% endX %}` 写法 |
|------|:--:|------|------|
| **块级** | 无前缀 | 多段落 + 表格 + 图片等任意内容 | 各自独占一个段落 |
| **段落区** | `{%p %}` | 多个纯文字段落，**不能含表格** | 各自独占一个段落（回车） |
| **行内** | `{%r %}` | 段落中的一段文字（单个 `<w:r>`） | 在同一段落内 |

> **关键**：{%p %} 和 {%r %} 的区别——{%p if %} 控制整段或多段的显隐，标签所在段落会被删除。{%r if %} 只在行内控制几个字的显隐，不产生新段落，保留首行缩进等样式。

## B. 变量与标签速查

| 语法 | 说明 | 示例 |
|------|------|------|
| `{{ Sheet.字段 }}` | 单值变量 | `{{ 全局.公司名称 }}` |
| `{{ i.列标题 }}` | 循环内变量 | `{{ i.股东名称 }}` |
| `{{ i.字段 \| num }}` | 带过滤器的变量 | `{{ i.认缴出资额 \| num }}` |
| `{% for i in Sheet %}` ... `{% endfor %}` | **块级循环**（多段落+表格） | `{% for i in 项目基本情况 %}` |
| `{% if 条件 %}` ... `{% endif %}` | **块级条件**（含表格区块） | `{% if 有表格内容 %}` |
| `{%p if 条件 %}` ... `{%p endif %}` | **段落区条件**（纯文字，多段落） | `{%p if 税率政策.有三免三减半 %}` |
| `{%p if 条件 %}` ... `{%p else %}` ... `{%p endif %}` | 段落区条件-分支 | |
| `{%r if 条件 %}文字{%r endif %}` | **行内条件**（一句话中几个字） | `{%r if 折扣 > 0 %}(折扣后){%r endif %}` |
| `{%tr for i in Sheet %}` ... `{%tr endfor %}` | 表格行循环 | `{%tr for i in 历史沿革.股权结构 %}` |
| `{%tr for k in i.子表 %}` ... `{%tr endfor %}` | 嵌套行循环 | `{%tr for k in i.epc金额 %}` |
| `{%tc for i in Sheet %}` ... `{%tc endfor %}` | 表格列循环 | `{%tc for i in 项目基本情况 %}` |

## C. 过滤器速查

| 过滤器 | 输入 | 输出 | 0值处理 |
|--------|------|------|:----:|
| `\| num` | 1234567.89 | 1,234,567.89 | 空（默认2位小数） |
| `\| num(0)` | 1234567.89 | 1,234,568 | 0位小数 |
| `\| num(2, '万')` | 12345678 | 1,234.57 | 除以万，2位小数 |
| `\| percent` | 0.25 | 25.00% | 空 |
| `\| date` | 2025-01-15 | 2025年01月15日 | 空，预设 `'short'` `'month'` `'chinese'` |
| `\| date('short')` | 2025-01-15 | 2025/01/15 | |
| `\| date('month')` | 2025-01-15 | 2025年01月 | |
| `\| date('chinese')` | 2025-01-15 | 二〇二五年一月十五日 | |
| `\| int` | 123.45 | 123 | 截断取整 |
| `\| paragraphs` | 第一行\n第二行 | Word段落换行 | 保留首行缩进 |
| `\| default_dash` | None/"" | "" | — |
| `\| default('-')` | None/"" | - | — |

> `\| money` 保留为 `\| num` 别名，向后兼容。

## D. 条件语法

| 语法 | 层级 | 说明 |
|------|------|------|
| `{% if 变量 %}` ... `{% endif %}` | 块级 | 控制含表格的区块显隐 |
| `{%p if 变量 %}` ... `{%p endif %}` | 段落区 | 变量为真时显示 |
| `{%p if not 变量 %}` ... `{%p endif %}` | 段落区 | 变量为假时显示 |
| `{%p if 变量 == '值' %}` ... `{%p endif %}` | 段落区 | 等于判断 |
| `{%p if 数量 > 0 %}` ... `{%p endif %}` | 段落区 | 数值比较 |
| `{%p else %}` | 段落区 | 否则分支 |
| `{%r if 条件 %}文字{%r endif %}` | 行内 | 一行中几个字的显隐 |

## E. 循环语法

### 行循环
```
{%tr for i in Sheet名 %}
{{ i.列1 }}  {{ i.列2 | num }}
{%tr endfor %}
```

**规则**：
- `{%tr for %}` 和 `{%tr endfor %}` 各占独立一行
- 循环变量统一用 `i`
- 循环内可以嵌套 `{%r if %}` 行内条件

### 嵌套行循环（列表内子表）
```
{% for i in EPC合同 %}
  EPC：{{ i.承包单位 }}
  ┌────────┬──────────┬──────────┐
  │ 序号   │ 合同分项  │ 金额     │
  ├────────┼──────────┼──────────┤
  {%tr for k in i.epc金额 %}
  │ {{ k.序号\|int }} │ {{ k.合同分项 }} │ {{ k.分项金额\|num }} │
  {%tr endfor %}
  └────────┴──────────┴──────────┘
{% endfor %}
```

### 段落循环
```
{% for i in Sheet名 %}

{{ i.属性1 }}
描述：{{ i.属性2 }}

{% endfor %}
```

### 列循环
```
┌──────┬────────────────────────────────┬──────┐
│ 固定 │ {%tc for i in Sheet名 %}          │ 固定 │
│      │ {{ i.字段 }}                    │      │
│      │ {%tc endfor %}                 │      │
└──────┴────────────────────────────────┴──────┘
```

**规则**：
- `{%tc for %}` 和 `{%tc endfor %}` 各占独立一格
- 中间的 `{{ i.字段 }}` 占一格，是动态复制的模板列
- N 个元素 → 生成 N 列（替换掉标签格+模板格）
- 可与 `{%tr for %}` 嵌套实现行列双动态
- **自动继承**：首行声明后，后续行无需重复书写 `{%tc for %}`

## G. CLI 参数速查

```bash
python -m src -d 数据.xlsx -t 模板.docx -o 输出.docx
```

| 参数 | 简写 | 说明 |
|------|:--:|------|
| `--data` | `-d` | 数据文件 |
| `--template` | `-t` | 模板文件 |
| `--output` | `-o` | 输出文件 |
| `--batch` | `-b` | 批量：数据目录 |
| `--output-dir` | | 批量：输出目录 |
| `--strict` | | 未定义变量时报错退出 |
| `--no-check` | | 跳过变量检查 |
| `--validate` | | 启用 Schema 验证 |
| `--strict-validate` | | 验证失败时报错退出 |
| `--schema` | | Schema 文件路径 |
| `--check-syntax` | | 渲染前检查语法 |
| `--report-unused` | | 报告未使用的字段 |
| `--verbose` | `-v` | 终端显示 DEBUG 输出 |
| `--quiet` | `-q` | 终端仅显示警告以上 |
| `--log-file` | | 完整日志写入文件（默认自动写 `logs/generate_*.log`） |
| `--no-log` | | 禁用文件日志 |
