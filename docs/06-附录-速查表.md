## A. 变量语法

| 语法 | 说明 | 示例 |
|------|------|------|
| `{{ Sheet.字段 }}` | 单值变量 | `{{ 全局信息.公司名 }}` |
| `{{ i.列标题 }}` | 循环内变量 | `{{ i.股东 }}` |
| `{{ i.列标题 \| money }}` | 带过滤器的变量 | `{{ i.认缴金额 \| money }}` |
| `{%p if 条件 %}` ... `{%p endif %}` | 段落级条件 | `{%p if 基本情况.有职工信息 %}` |
| `{%p if 条件 %}` ... `{%p else %}` ... `{%p endif %}` | 段落级条件-分支 | |
| `{%tc for i in list %}` ... `{%tc endfor %}` | 列循环（横向展开单元格） | |
| `{%tr for i in Sheet %}` ... `{%tr endfor %}` | 表格行循环 | 见下文 |

## C. 条件语法

| 语法 | 说明 |
|------|------|
| `{%p if 变量 %}` | 变量为真时显示 |
| `{%p if not 变量 %}` | 变量为假时显示 |
| `{%p if 变量 == '值' %}` | 等于判断 |
| `{%p if 变量 != '值' %}` | 不等于判断 |
| `{%p if 数量 > 0 %}` | 数值比较 |
| `{%p else %}` | 否则分支 |
| `{%p endif %}` | 条件结束 |
| `{%r if 条件 %}文字{%r endif %}` | 行内条件 |

## D. 循环语法

### 行循环
```
{%tr for i in Sheet名 %}
{{ i.列1 }}  {{ i.列2 | money }}
{%tr endfor %}
```

**规则**：
- `{%tr for %}` 和 `{%tr endfor %}` 各占独立一行
- 循环变量统一用 `i`
- 循环内可以嵌套 `{%r if %}` 行内条件

### 列循环
```
┌──────┬────────────────────────────────┬──────┐
│ 固定 │ {%tc for col in columns %}       │ 固定 │
│      │ {{ col }}                      │      │
│      │ {%tc endfor %}                 │      │
└──────┴────────────────────────────────┴──────┘
```

**规则**：
- `{%tc for %}` 和 `{%tc endfor %}` 各占独立一格
- 中间的 `{{ col }}` 占一格，是动态复制的模板列
- N 个元素 → 生成 N 列（替换掉标签格+模板格）
- 可与 `{%tr for %}` 嵌套实现行列双动态

## F. CLI 参数速查

```bash
python -m src -d 数据.xlsx -t 模板.docx -o 输出.docx
```

| 参数 | 简写 | 说明 |
|------|:--:|------|
| `--data` | `-d` | 数据文件 |
| `--template` | `-t` | 模板文件 |
| `--output` | `-o` | 输出文件 |
| `--batch` | `-b` | 批量：数据目录 |
| `--output-dir` | | 批量：输出目录 |
| `--strict` | | 未定义变量时报错退出 |
| `--no-check` | | 跳过变量检查 |
| `--validate` | | 启用 Schema 验证 |
| `--strict-validate` | | 验证失败时报错退出 |
| `--schema` | | Schema 文件路径 |
| `--check-syntax` | | 渲染前检查语法 |
| `--report-unused` | | 报告未使用的字段 |
| `--verbose` | `-v` | 终端显示 DEBUG 输出 |
| `--quiet` | `-q` | 终端仅显示警告以上 |
| `--log-file` | | 完整日志写入文件 |
