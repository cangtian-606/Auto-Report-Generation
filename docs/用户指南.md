## 二、Excel数据填写规范

### 2.1 基本格式

Excel文件包含一个或多个Sheet，每个Sheet对应模板中的不同数据域。

**Sheet命名规则**：

| Sheet名称 | 用途 | 数据格式 |
|-----------|------|---------|
| `date.xxx` | 键值对（全局变量、文字说明、条件标志等） | 键值对 |
| `form.xxx` | 循环数据（表格） | 表格 |

### 2.2 键值对Sheet格式

适用于单个变量的值：

| 字段编码 | 值 |
|----------|-----|
| date.公司名 | 公司名称 |
| date.日期 | 2025-01-15 |
| date.备注 | 这是备注 |
| date.显示详情 | TRUE |

**规则**：
- 第一行是标题（自动跳过）
- A列：字段编码（必须与模板变量名匹配）
- B列：值
- 布尔值用 `TRUE`/`FALSE`（大写）
- 数值直接填写数字
- 文本直接填写文字

### 2.3 表格Sheet格式

适用于循环生成的数据：

**Sheet名称**：`form.名称`（对应模板中的 `{{ form.名称 }}`）

| 名称 | 数量 | 单价 | 金额 |
|------|------|------|------|
| 产品A | 100 | 50.00 | 5000.00 |
| 产品B | 50 | 80.00 | 4000.00 |

**规则**：
- 第一行是列标题（对应变量属性名）
- 后续行是数据
- 数值列直接填数字
- 空值留空

## 三、变量命名参考

### 3.1 两种数据类型

| 类型 | Sheet前缀 | 模板引用 | 用途 |
|------|-----------|---------|------|
| 键值对 | `date.xxx` | `{{ date.字段 }}` | 单个变量值 |
| 表格 | `form.xxx` | `{%tr for item in form.名称 %}` | 循环数据列表 |

### 3.2 过滤器使用

在变量后加 `| 过滤器` 进行格式化：

| 过滤器 | 效果 | 示例输入 | 输出 |
|--------|------|---------|------|
| `money` | 千分位金额 | `1234567.89` | `1,234,567.89` |
| `percent` | 百分比 | `0.1567` | `15.67%` |
| `default('-')` | 空值默认 | `''` | `-` |

**使用示例**：
```
金额：{{ form.金额 | money }}元
比例：{{ form.比例 | percent }}
备注：{{ date.备注 | default('无') }}
```

## 四、命令执行

### 4.1 单文件生成

```bash
python -m src.renderer --data 数据.xlsx --template 模板.docx --output 输出.docx
```

**简化写法**：
```bash
python -m src.renderer -d 数据.xlsx -t 模板.docx -o 输出.docx
```

### 4.2 批量生成

```bash
python -m src.renderer --batch 数据目录/ --template 模板.docx --output-dir 输出目录/
```

**简化写法**：
```bash
python -m src.renderer -b 数据目录/ -t 模板.docx -o 输出目录/
```

### 4.3 常用选项

| 选项 | 说明 |
|------|------|
| `-d, --data FILE` | Excel数据文件 |
| `-o, --output FILE` | 输出文件路径 |
| `-b, --batch DIR` | 批量模式：数据文件目录 |
| `--strict` | 严格模式：未定义变量时报错 |
| `--no-check` | 跳过变量检查 |

- [ ] 金额/百分比格式是否正确

## 六、错误处理

### 常见错误

| 错误信息 | 原因 | 解决方法 |
|---------|------|---------|
| `未定义变量: xxx` | 数据中缺少该字段 | 在Excel中添加该字段 |
| `找不到Sheet: xxx` | Sheet名称不匹配 | 检查Sheet命名 |
| `渲染失败` | 模板语法错误 | 检查模板变量格式 |

### 数据校验

使用 `--no-check` 可跳过变量检查，强制生成：

```bash
python template_renderer.py -d data.xlsx -o output.docx template.docx --no-check
```

## 七、模板获取

业务人员通常不需要制作模板，模板由IT部门或模板管理员提供。

**获取模板后**：
1. 确认模板版本
2. 了解所需填写的数据字段
3. 按照本手册填写Excel数据
4. 执行生成命令

**如需新模板或修改现有模板**，请联系IT部门。

## 九、键值对Sheet详解

### 9.1 格式说明

适用于单个变量的值（如公司名称、日期、金额等）。

```
| 字段编码 | 值 |
|----------|-----|
| date.公司名 | 北京科技有限公司 |
| date.日期 | 2025-01-15 |
| date.金额 | 10000.50 |
```

**规则**：
- 第一行自动识别为标题并跳过
- A列：字段编码（`date.字段名` 或直接写字段名均可）
- B列：值
- 支持中文列名

### 9.3 数据类型

#### 字符串
```
| 字段编码 | 值 |
|----------|-----|
| date.名称 | 公司名称 |
```
直接填写文字。

#### 数字
```
| 字段编码 | 值 |
|----------|-----|
| date.年份 | 2025 |
| date.金额 | 10000.50 |
```
填写数字，会自动识别为数值类型。

#### 布尔值
```
| 字段编码 | 值 |
|----------|-----|
| date.显示 | TRUE |
| date.有附件 | FALSE |
```
使用 `TRUE`/`FALSE`（大写）。

#### 日期
```
| 字段编码 | 值 |
|----------|-----|
| date.日期 | 2025-01-15 |
| date.签订日期 | 2025年1月15日 |
```
支持多种日期格式。

## 十、表格型Sheet详解

### 10.1 格式说明

适用于循环数据（如商品列表、人员列表等）。

**Sheet命名**：`form.名称`（对应模板中的 `{{ form.名称 }}`）

```
| 名称 | 数量 | 单价 | 金额 |
|------|------|------|------|
| 产品A | 100 | 50.00 | 5000.00 |
| 产品B | 50 | 80.00 | 4000.00 |
```

**规则**：
- 第一行是列标题
- 列标题对应循环项的属性名
- 后续行是数据

### 10.2 命名对应关系

| Sheet命名 | 模板引用 |
|-----------|---------|
| `form.商品列表` | `{%tr for item in form.商品列表 %}` |
| `form.人员信息` | `{%tr for person in form.人员信息 %}` |
| `form.项目明细` | `{%tr for p in form.项目明细 %}` |

### 10.3 多Sheet对应

一个Excel可以有多个表格型Sheet：

```
Sheet: form.产品列表  →  {{ form.产品列表 }}
Sheet: form.服务列表  →  {{ form.服务列表 }}
```

## 十一、数据映射规则

### 11.1 自动映射

系统根据字段编码自动分配：

```
date.公司名  →  context['date']['公司名']
date.日期    →  context['date']['日期']
```

### 11.2 表格数据映射

表格Sheet直接映射为数据列表：

```
Sheet: form.商品列表 (名称, 数量, 单价, 金额)
        ↓
context['form']['商品列表'] = [
    {名称: "产品A", 数量: 100, 单价: 50.00, 金额: 5000.00},
    {名称: "产品B", 数量: 50, 单价: 80.00, 金额: 4000.00}
]
```

### 11.3 布尔值转换

| Excel填写 | Python值 | 模板判断 |
|-----------|---------|---------|
| `TRUE` | `True` | `{% if date.显示 %}✓{% endif %}` ✓ |
| `FALSE` | `False` | `{% if date.显示 %}` ✗ |
| `true` | `True` | ✓ |
| `1` | `True` | ✓ |
| `0` | `False` | ✗ |

## 十二、数据验证

### 12.1 必填字段

在Excel中添加必填检查，确保关键数据不缺失。

### 12.2 数据范围

对于数值字段，可添加数据验证：

```
数量：1-1000
比例：0-1
金额：>0
```

### 12.3 格式统一

统一日期、数字的填写格式，避免歧义。
