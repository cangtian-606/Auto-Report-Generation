# 数据格式规范

> 本文档是数据格式的**唯一事实源**。其他文档通过引用本文档来避免重复。

---

## 一、数据文件整体结构

系统支持 **Excel**（`.xlsx` / `.xls`）和 **YAML**（`.yaml` / `.yml`）两种格式，根据扩展名自动识别。

### Excel 结构

```
Excel 文件
├── Sheet: 全局信息      ← 键值对
├── Sheet: 基本情况      ← 键值对
├── Sheet: 股东出资      ← 循环表格
└── Sheet: 投资明细      ← 循环表格
```

### YAML 结构

```yaml
全局信息:
  公司名: 深圳市创新科技有限公司
  日期: "2025-01-15"
基本情况:
  注册资本: 1000
  有职工信息: true
股东出资:
  - 股东: 张伟
    认缴金额: 600
  - 股东: 李芳
    认缴金额: 250
```

## 二、两种数据类型

| 类型 | Excel 表现 | 模板引用方式 | 用途 |
|------|-----------|-------------|------|
| 键值对 | 两列：字段编码、值 | `{{ Sheet名.字段名 }}` | 单个值（名称、日期、金额等） |
| 循环表格 | 列标题 + 数据行 | `{%tr for i in Sheet名 %}` | 行数不固定的列表 |

---

## 三、键值对格式

### Excel

| 字段编码 | 值 |
|----------|-----|
| 公司名 | 深圳市创新科技有限公司 |
| 注册资本 | 1000 |
| 成立日期 | 2018-06-15 |
| 有职工信息 | TRUE |

**规则**：
- 第一行是标题行，系统自动跳过
- A 列写字段编码（只写字段名，Sheet 名自动作为前缀）
- B 列写值
- 布尔值用 `TRUE` / `FALSE`（不区分大小写）
- 数值直接填数字，文本直接填文字

### YAML

```yaml
全局信息:
  公司名: 深圳市创新科技有限公司
  注册资本: 1000
  成立日期: "2018-06-15"
  有职工信息: true
```

## 四、表格（循环列表）格式

### Excel

| 股东 | 认缴金额 | 认缴比例 |
|------|---------|---------|
| 张伟 | 600     | 0.60    |
| 李芳 | 250     | 0.25    |
| 王强 | 150     | 0.15    |

**规则**：
- 第一行是列标题
- 每行一条数据
- 列标题即字段名，模板中用 `{{ i.列标题 }}` 引用
- 循环变量统一使用 `i`

### YAML

```yaml
股东出资:
  - 股东: 张伟
    认缴金额: 600
    认缴比例: 0.60
  - 股东: 李芳
    认缴金额: 250
    认缴比例: 0.25
```

## 五、命名规范

### Sheet 命名

使用中文语义名称，见名知义：

| Sheet 名 | 用途 |
|----------|------|
| 全局信息 | 封面/全局字段 |
| 基本情况 | 基本信息字段 |
| 财务数据 | 财务相关字段 |
| 股东出资 | 股东列表 |
| 投资明细 | 投资列表 |

### 字段命名原则

1. 使用中文，见名知义
2. 同一报告内变量名保持唯一
3. 相似概念归入同一 Sheet
4. 布尔控制字段以"有"或"是否"开头（如 `有职工信息`、`有附件`）

### 变量引用语法

```
Sheet: 全局信息
  字段编码: 公司名   → 模板变量: {{ 全局信息.公司名 }}

Sheet: 股东出资
  列标题: 股东       → 循环变量: {{ i.股东 }}
```

## 六、嵌套数据

通过 Sheet 命名中的 `.` 表达父子关系。

### Excel

**父 Sheet**：`项目公司`（含子表的父列表）

| 公司简称 | 注册资本 |
|---------|---------|
| 重庆晟和泰 | 1000 |
| 安徽富军 | 800 |

**子 Sheet**：`项目公司.股东出资`

| _parent_公司简称 | 股东 | 认缴金额 |
|-----------------|------|---------|
| 重庆晟和泰 | 张伟 | 600 |
| 重庆晟和泰 | 李芳 | 250 |
| 安徽富军 | 王强 | 800 |

**规则**：
- `父Sheet.子Sheet`：点号分隔
- 子表首列必须以 `_parent_` 开头，值匹配父表的同名列
- 键值对子表自动挂载为 dict，表格子表自动挂载为 list

### YAML

```yaml
项目公司:
  - 公司简称: 重庆晟和泰
    股东出资:
      - 股东: 张伟
        认缴金额: 600
```

YAML 天然支持嵌套。

## 七、过滤器速查

> 这是过滤器的**唯一定义源**。修改过滤器逻辑请更新 `src/render/filters.py`。

| 过滤器 | 输入示例 | 输出 | 说明 |
|--------|---------|------|------|
| `\| money` | 1234567.89 | 1,234,567.89 | 千分位金额，0 / 空 → "" |
| `\| percent` | 0.25 | 25.00% | 小数自动乘 100，0 → "" |
| `\| num` | 1234567 | 1,234,567 | 千分位数字，整数不显示小数 |
| `\| date` | 2025-01-15 | 2025年01月15日 | 支持 `-`、`/`、年月日 格式 |
| `\| int` | 123.45 | 123 | 截断取整 |
| `\| default_dash` | None/"" | "" | 空值显示为空，非空返回字符串 |
| `\| default('-')` | None/"" | - | 空值用自定义字符串替换 |

**重要**：`money`、`percent`、`num` 对 0 值返回空字符串。如需显示 "0"，直接输出数值不加过滤器。

## 八、布尔控制字段

用于控制条件显示：

| Excel 值 | 模板用法 | 效果 |
|---------|---------|------|
| TRUE | `{%p if 基本情况.有职工信息 %}` | 显示该块 |
| FALSE | `{%p if 基本情况.有职工信息 %}` | 隐藏该块 |
| 不填 | `{%p if 基本情况.有职工信息 %}` | 隐藏该块 |

TRUE / FALSE 不区分大小写，前后空格自动去除。
