# Triage Labels

技能使用五种标准分诊角色标签。此文件映射这些角色到本仓库实际使用的标签字符串。

| 技能中使用的标签 | 本仓库标签 | 含义 |
|-----------------|-----------|------|
| `needs-triage` | `needs-triage` | 维护者需要评估此 issue |
| `needs-info` | `needs-info` | 等待提交者补充更多信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，AI agent 可独立接手 |
| `ready-for-human` | `ready-for-human` | 需要人工实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色（如"打上 ready-for-agent 标签"），使用此表中对应的标签字符串。

如需自定义标签词汇，直接编辑右列即可。
